◯ Lancamento Coberto / Cash-Secured PUT
Avalia se o premio recebido na venda da opcao supera X x CDI do periodo. Replica fielmente a planilha do Stormer.
Parametros
Voce ja tem as acoes. Vende uma CALL recebendo premio. Se o preco fechar abaixo do strike no vencimento, mantem as acoes + premio.
Preco do ativo (R$) spot
Strike da opcao (R$) strike
Premio recebido (R$) premium
Dias uteis ate vencimento 252 d/ano
CDI anual (%) benchmark
Retorno minimo desejado (X x CDI) multiplicador
Logica: a taxa do trade (premio sobre capital travado) deve superar X x CDI ajustado ao prazo.
taxa = strike / (spot - premium) - 1
cdi_periodo = (1 + cdi) ^ (dias/252) - 1
veredito = ADEQUADA se cdi_periodo * X < taxa
Resultado
Preencha os parametros e clique em "Avaliar Trade" para ver o veredito.
